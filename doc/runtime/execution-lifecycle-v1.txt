Materialize Plan

RUNBOOK преобразуется в plan:

- нормализация  
- валидация  
- сохранение plan.json  

Важно:

Каждый run создаёт plan artifact до выполнения.


Execution

Выполнение происходит через run-plan.

Runtime больше не интерпретирует runbook напрямую.


History

History сохраняет:

- trace  
- outbox  
- planRelPath  

planRelPath указывает на plan.json.
