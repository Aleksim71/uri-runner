3 Plan Compile

7 Baseline Verification
8 Outbox Packaging


Intake Gate

Перед фазой Inbox Intake URI выполняет фильтрацию входящих пакетов.

Назначение:

принимать только пакеты, адресованные URI.

Правило:

receiver == uri → ACCEPT  
receiver != uri → IGNORE  
receiver отсутствует → REJECT

Intake Gate не создаёт:
- trace
- outbox
- execution events

Это чистый фильтр входящих данных.
