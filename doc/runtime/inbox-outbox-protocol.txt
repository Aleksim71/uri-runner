Протокол Inbox–Outbox


Внешний выход

outbox.zip

Содержимое:

outbox.json  
provided files  


Внутренний output

trace.json  
events.jsonl  
plan.json (A13)  


A13 ДОПОЛНЕНИЕ

Каждый run создаёт plan.json:

- сохраняется в runtime artifacts  
- связан с history через planRelPath  


A14 ДОПОЛНЕНИЕ

Если compiled plan структурно некорректен:

- execution не запускается  
- outbox формируется как ошибка  

Тип ошибки:

PLAN_SCHEMA_INVALID  


Outbox contract

outbox.zip всегда содержит:

- outbox.json  
- provided files (если есть)  


exitCode:

0 → success  
1 → error  


Получатель (receiver)

receiver: uri


Правило:

URI обрабатывает только пакеты с receiver == uri
