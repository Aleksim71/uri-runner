# Context Metadata (URI Context OS)

Контекстный файл состоит из:

META + BODY

META хранит системные поля:

-   context_id
-   context_name
-   project
-   context_type
-   state
-   owner
-   group
-   permissions
-   created_at
-   updated_at
-   parent
-   level

Дополнительные поля:

-   goal_ref
-   goal_test_ref
-   freeze_ref
-   rules_ref
-   rights_ref
-   deadline_ref
-   summary_ref

META содержит только системную информацию и основные ссылки.
