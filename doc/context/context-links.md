# Context Links

Ссылка в URI состоит из двух полей:

id name

id --- идентификатор контекста name --- имя ссылки внутри текущего
контекста

Пример логической структуры:

link: id: ctx_market_research name: market_data

Текстовая форма:

\[\[ctx_market_research \| market_data\]\]

link_name не ограничивается словарём.
